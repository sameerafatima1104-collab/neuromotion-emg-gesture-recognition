# NeuroMotion – EMG-Based Gesture Recognition System

Real-time EMG gesture recognition system using ESP32 and machine learning classification.

## Overview
NeuroMotion is a real-time embedded + machine learning system that recognizes hand gestures using EMG sensor signals.

## Features
- Real-time EMG signal acquisition
- Signal preprocessing and filtering
- Feature extraction pipeline
- ML-based gesture classification
- ESP32 hardware integration

## Tech Stack
- Java
- ESP32
- Machine Learning
- Signal Processing

## Results
- Achieved ~88% classification accuracy
- Improved consistency across repeated trials
